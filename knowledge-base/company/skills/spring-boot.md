# Spring Boot Backend Development

## Overview

Core skill: building production-grade Java backends using Spring Boot 3.x.

## Key Areas

- **REST API design** — versioning, DTOs, exception handling, validation
- **JPA / Hibernate** — entity mapping, lazy loading, N+1 avoidance
- **Spring Security** — JWT authentication, role-based access, filter chains
- **Testing** — JUnit 5, Mockito, @SpringBootTest, Testcontainers
- **Observability** — Spring Actuator, Micrometer, structured logging

## Versions Used

| Library | Version |
|---------|---------|
| Spring Boot | 3.3.x |
| Java | 17 LTS |
| Hibernate | 6.x |
| Spring Security | 6.x |
