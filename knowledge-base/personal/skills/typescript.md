# TypeScript & Node.js

## Type System

- Generics, conditional types, mapped types, template literal types
- Utility types: `Partial`, `Required`, `Record`, `Pick`, `Omit`, `Readonly`
- Type narrowing, discriminated unions, exhaustive checks

## Async Patterns

- `async/await`, `Promise.all`, `Promise.allSettled`
- Streaming with `ReadableStream` and async iterators
- Error handling with typed custom error classes

## Node.js

- ESM modules (`import`/`export`, `type: "module"`)
- File system, streams, child processes
- Tooling: `tsx`, `ts-node`, `esbuild`, `tsc`
